AI Integration in Wearables for Clot Monitoring

The Clinical Challenge (The Issue)

Standard machine learning models for clinical wearables inherently optimize for global classification accuracy. This objective function inadvertently suppresses the detection of statistically rare, yet life-threatening, physiological anomalies like blood clots—creating the "Fatal Negative" paradox where a model appears 99% accurate while missing every critical emergency.

The fail-safe Proposal (The Solution)

This project proposes a Spatial-Relational-Temporal Hybrid architecture (CNN-Transformer-LSTM) that prioritizes clinical safety over raw accuracy. By abandoning passive argmax classification in favor of an active "Safety-First" Bayesian probability gate, the system ensures 100% Emergency Recall for unseen patient cohorts.

15. Mathematical Framework (Clinical Optimization)

To solve the "Fatal Negative" paradox, the model is optimized using a Weighted Focal Loss function that mathematically penalizes errors on rare clinical events (Emergencies) 5x more heavily than standard baseline activities.

\[ L(p_t) = -\alpha \cdot (1 - p_t)^\gamma \cdot \ln(p_t) \]

Weighted Focal Loss Objective

The Clinical Alpha ($\alpha$) Bias

Unlike standard Cross-Entropy, we utilize a class-specific weighting vector:
$\alpha_{SAFE} = 1.0$ | $\alpha_{WARNING} = 2.5$ | $\alpha_{EMERGENCY} = 5.0$

The $\gamma = 2.0$ parameter forces the 128-dimensional latent space to focus on "hard" examples near the decision boundary, reducing model over-confidence on majority Baseline signals.

Project Mathematical Registry (Actual Implemented Logic)

01. Sensor Fusion Normalization (Z-Score)
\[ z_i = \frac{x_i - \mu_j}{\sigma_j} \]

Standardizing BVP (Volts) and EDA ($\mu$S) into a shared unit-less manifold for the 1D-CNN encoder.

02. Dimensionality Reduction (PCA)
\[ C = \frac{1}{N-1} \sum_{i=1}^{N} (x_i - \bar{x})(x_i - \bar{x})^T \]

The covariance matrix $C$ is decomposed to identify the top 3 eigenvectors (PCs) for 3D visualization.

03. Prediction Entropy (Shannon)
\[ H(y|x) = -\sum_{c \in C} p(c|x) \log p(c|x) \]

Quantifying average information surprise across T=50 stochastic dropout passes.

04. Generative Penalty (GP)
\[ L_{GP} = \lambda \mathbb{E}_{\hat{x}} [(\|\nabla_{\hat{x}} D(\hat{x})\|_2 - 1)^2] \]

Enforcing 1-Lipschitz continuity in the critic to stabilize synthetic high-risk sample generation.

16. Formal Methodology (Algorithm Blocks)

Rigorous definition of the internal inference loops and the fail-safe gating logic.

Algorithm 1 Bayesian MC-Dropout
Input: Trained Model M, Input X, Iterations T=50 Output: Mean Prob P, Uncertainty V 1. Initialize Prob_List = [] 2. For t = 1 to T: 3. Enable Dropout in M 4. p_t = M_forward(X) 5. Append p_t to Prob_List 6. End For 7. P = Mean(Prob_List) 8. V = Variance(Prob_List) 9. Return P, V
Calculating posterior predictive distribution via stochastic passes.
Algorithm 2 Clinical Gating Logic
Input: Probabilities P=[P_s, P_w, P_e] Output: Gated Class C_g 1. Define Threshold Tau = 0.1094 2. C_std = argmax(P) 3. If C_std == SAFE Then: 4. If (P_w + P_e) > Tau Then: 5. C_g = WARNING // Safety Promotion 6. Else: 7. C_g = SAFE 8. Else: 9. C_g = C_std 10. Return C_g
Probabilistic override ensuring high-sensitivity emergency detection.
