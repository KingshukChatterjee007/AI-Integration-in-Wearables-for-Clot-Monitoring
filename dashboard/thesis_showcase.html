03. Mathematical Foundation

Implementing Weighted Focal Loss to penalize minority-class omissions and Bayesian MC-Dropout for real-time uncertainty quantification.

\[ L(p_t) = -\alpha_t \cdot (1 - p_t)^\gamma \cdot \ln(p_t) \]

Algorithm: MC-Dropout Inference Loop

Input: Model M, Data X, Iterations T=50 1. Init empty Probability Matrix P = [] 2. For t = 1 to T: 3. Set M to Inference Mode (Dropout ENABLED p=0.2) 4. p_t = M.forward(X) 5. Append p_t to P 6. End For 7. P_mean = Average(P); V = Variance(P) 8. Return P_mean, V
