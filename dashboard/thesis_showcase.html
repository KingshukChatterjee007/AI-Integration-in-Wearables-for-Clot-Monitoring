Clinical AI Monitoring Project

AI Integration in Wearables for Clot Monitoring

The Clinical Challenge (The Issue)

Standard machine learning models for clinical wearables inherently optimize for global classification accuracy. This objective function inadvertently suppresses the detection of statistically rare, yet life-threatening, physiological anomalies like blood clots—creating the "Fatal Negative" paradox where a model appears 99% accurate while missing every critical emergency.

The fail-safe Proposal (The Solution)

This project proposes a Spatial-Relational-Temporal Hybrid architecture (CNN-Transformer-LSTM) that prioritizes clinical safety over raw accuracy. By abandoning passive argmax classification in favor of an active "Safety-First" Bayesian probability gate, the system ensures 100% Emergency Recall for unseen patient cohorts.

07. Training Telemetry & Foundation

Loss Function: Weighted Focal Loss

\[ L(p_t) = -\alpha \cdot (1 - p_t)^\gamma \cdot \ln(p_t) \]

Alpha ($\alpha$) = 0.25 was used to balance class frequencies, while Gamma ($\gamma$) = 2.0 forced the model to focus on difficult, high-risk window signatures during training.
