# AI Integration in Wearables for Clot Monitoring - Data Explanation

## 📊 Overview of Preprocessed Data Files

Your preprocessing pipeline generated **5 specialized datasets** for AI-powered blood clot detection:

###  1. PPG_DATASET.CSV (60.84 MB)
- **Purpose**: Medical-grade cardiovascular data for training clot detection models
- **Content**: 2,576 patient records with 2,000 PPG signal channels + MI labels
- **Medical Significance**: PPG measures blood flow - critical for detecting circulation problems
- **Generated By**: Chunk-based loading with missing value imputation

###  2. SUBJECTS_INFO.CSV (0.01 MB)  
- **Purpose**: Demographics and physiological data for personalized risk assessment
- **Content**: 66 records with age, gender, BMI, blood pressure, heart rate changes
- **Medical Significance**: Demographics heavily influence clot risk (age, gender, obesity)
- **Generated By**: BMI calculations, physiological change computations, categorical encoding

###  3. SUBJECT_FEATURES.CSV (4.42 MB)
- **Purpose**: Real-time clot monitoring features from wearable sensors
- **Content**: 3,207 time windows × 128 features (ECG, PPG, temperature, motion)
- **Medical Significance**: Continuous monitoring enables early detection before symptoms
- **Generated By**: Statistical analysis of 5,000-point sensor data windows

### 4. INTEGRATED_FEATURES.CSV (4.61 MB)
- **Purpose**: Complete ML-ready dataset combining sensors + demographics
- **Content**: 3,207 windows × 145 features (128 sensor + 17 demographic + risk scores)
- **Medical Significance**: Combines real-time monitoring with clinical risk factors
- **Generated By**: Data merging + risk indicator calculations

###  5. ADVANCED_PPG_FEATURES.CSV (0.00 MB)
- **Purpose**: Specialized cardiovascular features for clinical validation
- **Content**: 20 windows × 22 cardiac features (signal quality, heart rate variability)
- **Medical Significance**: Heart rate variability indicates autonomic dysfunction
- **Generated By**: Advanced PPG signal processing and anomaly detection

##  Machine Learning Applications

### 1. Real-Time Clot Risk Prediction
- **Input**: `subject_features.csv` (wearable sensor data)
- **Model**: Time-series classification (LSTM/CNN)
- **Output**: Risk level every 10-20 seconds

### 2. Personalized Risk Assessment
- **Input**: `integrated_features.csv` (sensors + demographics)  
- **Model**: Ensemble methods (Random Forest + XGBoost)
- **Output**: Individual risk profile with confidence intervals

### 3. Cardiovascular Event Detection
- **Input**: `advanced_ppg_features.csv` (cardiac analysis)
- **Model**: Anomaly detection (Isolation Forest/Autoencoders)
- **Output**: Alert system for immediate medical attention

### 4. Clinical Validation
- **Input**: `ppg_dataset.csv` (medical-grade data)
- **Model**: Deep learning (CNN for signal classification)
- **Output**: Validation against known heart attack cases
